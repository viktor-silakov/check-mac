# Used MacBook Checker

Single-page checklist for inspecting a used MacBook before buying, with built-in browser tests (dead pixels, keyboard, camera, microphone, speakers, trackpad, port charging) and terminal commands whose output can be pasted back to auto-mark the checklist.

**Live:** https://viktor-silakov.github.io/check-mac/

Languages: English (default), Russian, Spanish, German. No build step, no dependencies — one `index.html`.
